# City 17 Serfers

A small open-source infinite runner inspired by **Subway Surfers** and set in the world of **Half-Life 2**. It is a fan project and uses only original code and placeholder graphics.

The game is written in Python using [Pygame](https://www.pygame.org/). Run the game with:

```bash
python3 main.py
```

Use the **Left** and **Right** arrow keys to move between lanes. Avoid obstacles and see how long you can last while running toward the distant Citadel of City 17!
