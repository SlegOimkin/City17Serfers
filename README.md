# City17 Serfers

A small endless runner built with [Three.js](https://threejs.org/) inspired by the world of *Half-Life 2*. The player runs toward the citadel of City 17 while dodging obstacles.

This project is a fan-made, open source tribute and does not use any assets from Half-Life 2.

## Running

Open `index.html` in a modern browser. Use the left and right arrow keys to change lanes and avoid red obstacles. Press `R` after crashing to restart.
